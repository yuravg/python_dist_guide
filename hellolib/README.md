# Hellolib

A simple Python package for hellolib messages.

## Usage

```bash
  # Create environment:
   python -m venv myenv

  # Building:
  python -m build

  # Activate environment
  source myenv/bin/activate

  # Install the package
  pip install .

  # Checking
  python -c "from hellolib.hello import say_hello; print(say_hello('Python Wheel'))"
```
